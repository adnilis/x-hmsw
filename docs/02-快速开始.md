# 快速开始

本指南将帮助您在 5 分钟内快速上手 x-hmsw 向量数据库。

## 安装

### 使用 Go Modules

```bash
go get github.com/adnilis/x-hmsw
```

### 从源码构建

```bash
git clone https://github.com/adnilis/x-hmsw.git
cd x-hmsw
go build
```

## 基础使用

### 1. 创建数据库

使用 QuickDB 创建一个向量数据库：

```go
package main

import (
    "fmt"
    "github.com/adnilis/x-hmsw/api"
)

func main() {
    // 创建向量数据库（默认配置）
    // 自动保存间隔：30秒
    db, err := api.NewQuick("./data")
    if err != nil {
        panic(err)
    }
    defer db.Close()

    fmt.Println("数据库创建成功！")
}
```

### 2. 自定义配置

```go

## 文本向量化

x-hmsw 内置了多种文本向量化方法，无需外部依赖。

### 1. 使用 TF-IDF 向量化

```go
package main

import (
    "fmt"
    "github.com/adnilis/x-hmsw/api"
    "github.com/adnilis/x-hmsw/embedding"
    "github.com/adnilis/x-hmsw/types"
)

func main() {
    db, _ := api.NewQuick("./data")
    defer db.Close()

    // 准备文档
    documents := []string{
        "机器学习是人工智能的一个分支",
        "深度学习使用多层神经网络",
        "自然语言处理处理文本数据",
    }

    // 创建 TF-IDF 向量化器
    config := embedding.DefaultTFIDFConfig()
    vectorizer := embedding.NewTFIDF(config)

    // 训练向量化器
    vectorizer.Fit(documents)

    // 将文档转换为向量
    vectors := make([]types.Vector, len(documents))
    for i, doc := range documents {
        vec := vectorizer.Transform(doc)
        vectors[i] = types.Vector{
            ID:     fmt.Sprintf("doc_%d", i),
            Vector: vec,
            Payload: map[string]interface{}{
                "content": doc,
            },
        }
    }

    // 插入向量
    db.Insert(vectors)

    // 搜索相似文档
    queryText := "神经网络学习"
    queryVec := vectorizer.Transform(queryText)
    query := types.Vector{Vector: queryVec}

    results, _ := db.Search(query, 3)
    for _, result := range results {
        fmt.Printf("相似度: %.4f, 内容: %s\n", result.Score, result.Payload["content"])
    }
}
```

### 3. 使用 OpenAI Embeddings

```go
package main

import (
    "fmt"
    "os"
    "github.com/adnilis/x-hmsw/api"
    "github.com/adnilis/x-hmsw/embedding"
    "github.com/adnilis/x-hmsw/types"
)

func main() {
    // 设置环境变量
    os.Setenv("OPENAI_API_KEY", "your-api-key")
    os.Setenv("OPENAI_BASE_URL", "https://api.openai.com/v1")
    os.Setenv("OPENAI_MODEL", "text-embedding-3-small")

    db, _ := api.NewQuick("./data")
    defer db.Close()

    // 创建 OpenAI Embedding 客户端
    config := embedding.DefaultConfig()
    client := embedding.NewOpenAIEmbedding(config)

    // 准备文档
    documents := []string{
        "机器学习是人工智能的一个分支",
        "深度学习使用多层神经网络",
        "自然语言处理处理文本数据",
    }

    // 批量获取向量
    vectors, err := client.CreateBatchEmbedding(documents)
    if err != nil {
        panic(err)
    }

    // 插入向量
    dbVectors := make([]types.Vector, len(documents))
    for i, vec := range vectors {
        dbVectors[i] = types.Vector{
            ID:     fmt.Sprintf("doc_%d", i),
            Vector: vec,
            Payload: map[string]interface{}{
                "content": documents[i],
            },
        }
    }
    db.Insert(dbVectors)

    // 搜索相似文档
    queryText := "神经网络学习"
    queryVec, _ := client.CreateEmbedding(queryText)
    query := types.Vector{Vector: queryVec}

    results, _ := db.Search(query, 3)
    for _, result := range results {
        fmt.Printf("相似度: %.4f, 内容: %s\n", result.Score, result.Payload["content"])
    }
}
```

## 聊天记录检索示例

x-hmsw 特别适合聊天记录的语义检索：

```go
package main

import (
    "fmt"
    "github.com/adnilis/x-hmsw/api"
    "github.com/adnilis/x-hmsw/embedding"
    "github.com/adnilis/x-hmsw/types"
)

type ChatMessage struct {
    ID        string
    Role      string
    Content   string
    Timestamp string
}

func main() {
    db, _ := api.NewQuick("./chat_data")
    defer db.Close()

    // 模拟聊天记录
    messages := []ChatMessage{
        {"msg_001", "user", "什么是机器学习？", "2024-01-01"},
        {"msg_002", "assistant", "机器学习是人工智能的一个分支...", "2024-01-01"},
        {"msg_003", "user", "深度学习和机器学习有什么区别？", "2024-01-02"},
        {"msg_004", "assistant", "深度学习是机器学习的子集...", "2024-01-02"},
    }

    // 创建 BM25 向量化器
    config := embedding.DefaultBM25Config()
    vectorizer := embedding.NewBM25(config)

    // 训练并转换
    documents := make([]string, len(messages))
    for i, msg := range messages {
        documents[i] = msg.Content
    }
    vectorizer.Fit(documents)

    // 插入向量
    vectors := make([]types.Vector, len(messages))
    for i, msg := range messages {
        vec := vectorizer.Transform(msg.Content)
        vectors[i] = types.Vector{
            ID:     msg.ID,
            Vector: vec,
            Payload: map[string]interface{}{
                "role":      msg.Role,
                "content":   msg.Content,
                "timestamp": msg.Timestamp,
            },
        }
    }
    db.Insert(vectors)

    // 搜索相关聊天记录
    queryText := "神经网络"
    queryVec := vectorizer.Transform(queryText)
    query := types.Vector{Vector: queryVec}

    results, _ := db.Search(query, 3)
    fmt.Printf("查询: %s\n", queryText)
    fmt.Println("相关聊天记录:")
    for _, result := range results {
        fmt.Printf("  [%s] %s (相似度: %.4f)\n",
            result.Payload["role"],
            result.Payload["content"],
            result.Score)
    }
}
```

## 高级配置

### 使用底层接口

如果需要更多控制，可以使用底层接口：

```go
package main

import (
    "github.com/adnilis/x-hmsw/interface"
    "github.com/adnilis/x-hmsw/types"
)

func main() {
    config := iface.Config{
        Dimension:      384,           // 向量维度
        IndexType:      iface.HNSW,    // 索引类型
        StorageType:    iface.Badger,  // 存储类型
        StoragePath:    "./data",      // 存储路径
        DistanceMetric: iface.Cosine,  // 距离度量
        MaxVectors:     1000000,       // 最大向量数
        M:              16,            // HNSW 参数
        EfConstruction: 200,           // HNSW 参数
    }

    db, err := iface.NewPureGoVectorDB(config)
    if err != nil {
        panic(err)
    }
    defer db.Close()

    // 使用数据库...
}
```

### 选择不同的索引类型

```go
// HNSW 索引（默认）
config.IndexType = iface.HNSW

// IVF 索引
config.IndexType = iface.IVF
config.NumClusters = 100  // 聚类数量
config.Nprobe = 10        // 搜索时检查的聚类数

// Flat 索引（暴力搜索）
config.IndexType = iface.Flat
```

### 选择不同的存储引擎

```go
// BadgerDB（默认）
config.StorageType = iface.Badger

// BBolt
config.StorageType = iface.BBolt

// PebbleDB
config.StorageType = iface.Pebble

// 内存存储
config.StorageType = iface.Memory

// 内存映射
config.StorageType = iface.MMap
```

### 选择距离度量

```go
// 余弦相似度（默认）
config.DistanceMetric = iface.Cosine

// 欧几里得距离
config.DistanceMetric = iface.L2

// 内积

```

## 常见问题

### Q: 如何选择索引类型？

**A**:
- **HNSW**: 适合大多数场景，提供高精度和高性能的平衡
- **IVF**: 适合大规模数据集（百万级以上），内存占用较低
- **Flat**: 适合小规模数据集（万级以下），提供精确结果

### Q: 如何选择存储引擎？

**A**:
- **BadgerDB**: 通用场景，性能均衡
- **BBolt**: 单机应用，简单可靠
- **PebbleDB**: 高写入场景
- **Memory**: 需要最快访问速度，数据可以丢失
- **MMap**: 需要持久化，但希望接近内存性能

### Q: 如何优化性能？

**A**:
1. 使用合适的索引类型和参数
2. 选择合适的距离度量
3. 使用向量压缩降低内存占用
4. 调整 HNSW 参数（M、EfConstruction）
5. 使用批量操作而非单条操作

### Q: 数据会自动保存吗？

**A**: 使用 QuickDB 时，默认每 30 秒自动保存一次。可以通过 `NewQuickWithConfig` 自定义保存间隔。
